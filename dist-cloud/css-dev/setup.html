3. Modifica CSS nel browser

Una volta aperta la pagina:

  1. Premi F12 per aprire Developer Tools
  2. Vai su Elements
  3. Seleziona un elemento
  4. Nella sezione Styles modifica direttamente i CSS
  5. Le modifiche sono immediate!

4. Salva le modifiche

Quando sei soddisfatto delle modifiche:

  1. Copia il CSS modificato dal browser
  2. Apri il file SCSS corrispondente in Cursor
  3. Incolla le modifiche
  4. Salva il file
  5. Esegui npm run build per compilare
